Open the deployed demo [https://nextjs-table-component.vercel.app/](https://nextjs-table-component.vercel.app/).

## Getting Started

First, install the dependencies:

```bash
npm i
# or
yarn
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
